数组

Array既是一个数组，也是一个字典。

语句 功能 示例
var a = new Array() 定义一个数组或者字典，定义字典最好使用 new Object
arr.push() 在数组末尾添加一个元素
arr.unshift() 在数组头部添加一个元素
arr.pop() 先进后出
arr.shift() 把数组的第一个元素从其中删除，并返回第一个元素的值
fruits.splice(pos, 1)
for (var key in dic){} 遍历
delete dic[“r”]; 删除一个KEY
delete dic.r 删除一个KEY
array.forEach(function(currentValue, index, arr), thisValue) 用于调用数组的每个元素，并将元素传递给回调函数,没有返回值，forEach无法跳出循环（使用return也不行，只能抛异常跳出）
[].map() 基本用法跟forEach方法类似，作用不难理解，“映射”嘛，也就是原数组被“映射”成对应新数组
[].some() 返回一个boolean，判断是否有元素是否符合func条件,返回true时终止遍历
[].every() 返回一个boolean，判断每个元素是否符合func条件,every 返回false时终止遍历,返回true继续遍历
array.length 数组的长度　｜
arrayObject.concat(arrayX,arrayX,……,arrayX) 返回一个新的数组,不改变原数组的值。该数组是通过把所有 arrayX 参数添加到 arrayObject 中生成的。如果要进行 concat() 操作的参数是数组，那么添加的是数组中的元素，而不是数组
Array.from() 从一个类似数组或可迭代对象中创建一个新的数组实例 `console.log(Array.from(‘foo’));// expected output: Array [“f”, “o”, “o”]console.log(Array.from([1, 2, 3], x => x + x));// expected output: Array [2, 4, 6]`
[…new Array(10).keys()] 序列化一个新数组
indexOf() 判断是否存在
  1. 判断JS对象是否拥有某属性两种方式，但稍有区别
