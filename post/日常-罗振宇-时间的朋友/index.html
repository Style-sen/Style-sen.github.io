1. 2015年

1.1. 互联网恐慌

  1. 用生物学思维重新理解商业；
  2. 何须转型，只要增长；

1.2. 资本寒冬

  1. 最适合创业的人是聪明、年轻、贫穷的人；
  2. 年轻人创业，失去的是锁链，赢得的是整个世界；
  3. 钱是世界上最聪明的东西，最负责任的东西；
  4. 经济越发达，资本越不值钱；
  5. 资本寒冬，不是大家不投了，是暂时不知道投什么资产合适；
  6. 寒武纪出现的两性繁殖，其实是为了获取物种的多样性；
  7. 资本和雄性一样，都花心；雌性，指的是互联网创业者；
  8. 商场上只讲力量;
  9. 王石真正值得尊敬，我真正等着看这场大戏的是，代表创业者、代表知识和人力资本的持有者，他怎么样通过堂堂正正的规则下的博弈，去引来浩瀚如大海一般的资本力量，去击溃门口的野蛮人;

1.3. 两只“妖股”

  1. 资本最恐惧的是坐吃山空，资本一定要找到新的出路;

1.4. O2O大战

  1. 程维说滴滴烧钱，其实并非他们的初衷;
  2. 金字塔收入结构被打破，现在更像是仙人掌模式，只要你有手艺，就能借助互联网渠道展业，手艺人的地位真真切切的在上升;
  3. 每一代连接技术都会产生这样的奇迹，互联网也是这样，必然产生一代红利，然而享有者未必是连接者;
  4. 互联网基础设施已经构建好了，要重新思考红利;

1.5. IP

  1. 可以摆脱单平台的束缚，对用户产生吸引力的事物；
  2. 让人们有存在感、是一种社交货币、是新型交易入口；
  3. 在某些特定领域，IP可以帮助粉丝制造存在感；
  4. 传统的交易入口，靠流量和性价比（如店面和集市），但人格化的交易入口正在崛起；
  5. 稀缺性是商品的核心价值；
  6. 让人觉得信任的“人”，是稀缺的；
  7. 罗辑思维是人格化的交易，粉丝购买力不会因为流量变大而提高；
  8. IP能够为商品提供稀缺性，满足和彰显消费者的“分别心”；

1.6. 腾讯和阿里

  1. 合法赚钱，是用户用钱投票创造的价值；
  2. 银行的重要职能是为“风险”去定价；
  3. 马云一直在努力重新定义阿里这家公司——关键在于对大数据的掌握和运用；
  4. 阿里的收购逻辑，也可以用数据获取的角度去分析理解；
  5. 阿里让世界更清晰地看到人；腾讯让人更高效地得到世界；
  6. 在商业上，你老了，就是新一代的基础设施，你已经不会用了。

1.7. 小米和华为

  1. 小米把硬件供应链撕开了一条口子，让智能手机门槛大大降低，值得尊敬。
  2. 线：拦路抢劫——小米的模式，是力求打造生态链，长久的赚钱；
  3. 点：坐地分赃——华为的模式，追求单品利润；
  4. 不去连线（进不去、守不住、跟不上），而是去造点；
  5. Apple，稳扎稳打，追求单品利润；
  6. 真诚才是成功的关键。

2. 2016年

2.1. 时间战场

  1. 宏观是我们必须接受的，微观才是我们能有所作为的；
  2. 现在所有的新兴产业本质上就是既要你的钱，还要你的命；
  3. 未来，在时间这个战场上，有两门生意会特别值钱：第一，就是帮别人省时间。第二，就是帮别人把省下来的时间浪费在那些美好的事物上。

2.2. 服务升级

  1. 今日头条，践行“母爱逻辑”，就是孩子要什么，妈给你更多，惯得你没样。
  2. 我的商业生涯要走在这个逻辑的反面，走在父爱逻辑里。首先我得站得更高、看得更远，我得知道什么是好东西，然后我也许态度粗暴冷峻地跟我的用户说，来，这个是好东西，我把它称之为叫“父爱算法”。在这个算法世界当中，一定有它的一席之地，最好的服务是给你还不知道的好东西。这是下一个消费升级的方向。

2.3. 智能革命

  1. 人工智能不是在复制人类，它是完全不同的另外一种存在。
  2. 人工智能不会提高玩家参与的门槛，是降低了参与门槛。
  3. 人工智能不仅是人的延伸，它是人的替代。
  4. 领导力和创造力。

2.4. 认知税

  1. 世界越来越破碎，而那些治愈破碎的力量就会变得越来越值钱。这个力量我们称之为共同的认知。
  2. 越来越稀缺的东西，就越来越值钱。这背后的逻辑和过去几十年的房地产业一模一样。

2.5. 共同体危机

  1. 有一个词，使用频率在2016年飙升了200倍——“后真相”。情绪的影响力已经超过了事实。
  2. 后真相（post-truth）的意思，不是没有真相，而是对这个世界来说，情绪的影响力已经超过了事实。
  3. 自律自律自律。

3. 2017年

3.1. 动车组脑洞

  1. 创新就是解决问题的能力。
  2. 创新必须是干一遍才知道。

3.2. 热带雨林脑洞

  1. 新物种，往往是高价值的连接器（货满满，快手）。
  2. 流量思维已经过时。
  3. 过度到超级用户思维（小生态）。
  4. 会员经济在企业和消费者之间建立了一种可持续可信任的正式关系。
  5. 不管你对普通用户有多好，请先关照超级用户的感受，面朝超级用户。做让用户长脸的事，不做让用户丢脸的事。彼此觉得荣耀。

3.3. 比特化脑洞

  1. 货正在一步步逼近人。
  2. 小米认知战。
  3. 做好自己，保持开放。
  4. 效率越来越高，分工越来越细。

5. 2019

  1. 贝聿铭：我从来没有考虑过这些问题，因为我一直沉浸在如何解决自己的问题中。
  2. 不解决一个个想象中的问题，回应一个个真实世界的挑战。
  3. 躬身入局。把自己放进去，把自己变成解决问题的关键变量。
  4. 容易赚的钱没了，往后大家都得做更辛苦的事。
  5. 别以为下沉市场就算更辛苦了，也别以为找细分需求就算更辛苦了，真正的机会，改造每一个只得被重构的传统产业，这才是一件具备创造力的苦差事。
  6. 每个人都能找到自己的着力点。往旁边看一看，迈出这么一小步，这就是你向上攀岩的着力点。
  7. 检验品牌，就是你愿意和他自拍。
  8. 利用中国红利和新基础设施创造一个世界级品牌。

5.1. 钱从哪里来

  1. 73%的劳动所得，个人辛苦赶出来的。27%理财所得。
  2. 人和人的连接，正在决定社会财富的创造、分配和转移。
  3. 你自己的本事，你和其他人连接的本事。
  4. 保险职业考验的是持续赢得用户尊重的能力。
  5. 本领可能会过时，资本可能会贬值，但是，人和人之间的连接是可以一直延续下去的。多和人连接保证没有坏处。
  6. 钱从自己的劳动里来，钱从更多的人和人的连接中来。
  7. 你是谁？你干啥？你为谁干？别人需要你吗？社会更好了吗？
